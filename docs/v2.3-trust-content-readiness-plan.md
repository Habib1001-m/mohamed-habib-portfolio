# v2.3-A — Trust Content Readiness Plan

## Decision

v2.3 starts with content readiness, not public trust activation.

The current portfolio may keep the lightweight motion layer active, but testimonials, booking, and case-study surfaces remain gated until real content and approval evidence are available.

## Current public state

- Testimonials: OFF
- Booking CTA: OFF
- Case studies: OFF
- Third-party analytics: OFF
- Motion layer: limited native motion only

## Readiness gates

### Testimonials

Public testimonials can only be enabled when all conditions are true:

1. At least 3 testimonials are available.
2. Each testimonial is approved for public use.
3. Each testimonial has a real source category.
4. No placeholder quote is shipped.
5. No anonymous quote is used unless explicitly approved.

### Booking

Booking can only be enabled when all conditions are true:

1. A real booking URL is approved.
2. Availability windows are defined.
3. Call purpose is clear.
4. Follow-up workflow is defined.
5. The CTA does not overpromise employment, partnership, or paid work outcomes.

### Case studies

Case studies can only be enabled when all conditions are true:

1. At least one case study status is changed from draft/restricted to ready.
2. The public case study contains proof-safe language.
3. Restricted project details remain hidden.
4. Screenshots and metrics are approved for public display.
5. The case study explains decision-making, not confidential implementation details.

## Non-negotiable rules

- Hidden is safe. Placeholder is unsafe.
- No fake social proof.
- No public trust claim without source.
- No private project leakage.
- No public booking link until workflow is approved.
- No turning on testimonials, booking, or case studies in the same commit that adds unreviewed content.

## Recommended sequence

1. v2.3-B — Testimonial Collection Schema + Empty Review Queue
2. v2.3-C — Case Study Readiness Audit
3. v2.3-D — Booking Workflow Decision
4. v2.3-E — Trust Surface Activation Review

## Go/No-Go

Current v2.3-A decision: NO-GO for public trust activation.

Approved next action: prepare content collection and review structures only.
